# DevOps & Kubernetes Masterclass

## Topics to Cover

### Docker
- `01_DockerBasics` – Dockerfile, build, run, push
- `02_DockerCompose` – Multi-container orchestration
- `03_DockerNetworking` – Bridge, Host, Overlay networks
- `04_DockerVolumes` – Persistent data for databases
- `05_MultiStageBuilds` – Smaller production images

### Kubernetes (K8s)
- `06_K8sBasics` – Pod, Node, Cluster concepts
- `07_Deployments` – Replicas, rolling updates, rollbacks
- `08_Services` – ClusterIP, NodePort, LoadBalancer
- `09_ConfigMaps` – Injecting config without rebuilding images
- `10_Secrets` – Encrypted environment variables in K8s
- `11_Ingress` – HTTP routing to multiple services
- `12_HPA` – Horizontal Pod Autoscaler (auto-scaling)
- `13_StatefulSets` – Databases and ordered pod management
- `14_PersistentVolumes` – Storage that survives pod deletion

### CI/CD Pipelines
- `15_GithubActions` – Build, test, deploy to K8s
- `16_Jenkins` – Classic enterprise CI/CD
- `17_ArgoCD` – GitOps: Git is the source of truth
- `18_Helm` – Kubernetes package manager (like npm for K8s)

### Observability in K8s
- `19_PrometheusGrafana` – Metrics collection and dashboards
- `20_Loki` – Log aggregation for Kubernetes
- `21_Jaeger` – Distributed tracing in K8s
