# Spring Security Masterclass

## Topics to Cover

### Authentication
- `01_BasicAuth` – Username/password, UserDetailsService
- `02_JwtAuth` – Sign, verify, refresh tokens
- `03_OAuth2Login` – Google, GitHub social login
- `04_KeycloakIntegration` – SSO enterprise auth server
- `05_SAMLIntegration` – Enterprise single sign-on

### Authorization
- `06_RoleBased` – ROLE_ADMIN, ROLE_USER, hasRole()
- `07_PermissionBased` – Fine-grained permissions (READ_ORDERS)
- `08_MethodSecurity` – @PreAuthorize, @PostAuthorize, @Secured
- `09_RowLevelSecurity` – User can only see THEIR data

### Attack Prevention
- `10_CSRF_Protection` – Cross-Site Request Forgery
- `11_CORS_Security` – Proper origin whitelisting
- `12_XSSPrevention` – Content Security Policy headers
- `13_SQLInjection` – Parameterized queries, JPA protection
- `14_BruteForce` – Login attempt limiting

### Advanced
- `15_SessionManagement` – Stateless vs stateful
- `16_SecurityEvents` – Audit login success/failure
- `17_TwoFactorAuth` – TOTP (Google Authenticator)
- `18_APIKeyAuth` – Service-to-service API key validation
