# Interview Preparation - Senior Java Engineer

## Topics to Cover (Each folder = One interview topic with Q&A + Code)

### Core Java (30 questions)
- `01_OOPConcepts` – Polymorphism, Abstraction, Encapsulation, Inheritance
- `02_CollectionsDeepDive` – HashMap internals, ConcurrentHashMap, TreeMap
- `03_ExceptionHandling` – Checked vs Unchecked, try-with-resources
- `04_StringInterning` – String pool, == vs equals, StringBuilder
- `05_JavaMemoryModel` – JVM Heap, Stack, GC behavior
- `06_ThreadSafety` – Race conditions, volatile, synchronized
- `07_Java8Features` – Streams, Optional, Lambda, Date API

### Spring Boot (25 questions)
- `08_SpringIOCDI` – Bean lifecycle, circular deps, lazy loading
- `09_SpringAOP` – Proxies, pointcuts, advice types
- `10_TransactionManagement` – Propagation, isolation, rollback rules
- `11_SpringSecurity` – Filter chain order, JWT internals
- `12_SpringDataJPA` – N+1 problem, fetch types, caching

### Microservices (20 questions)
- `13_ServiceDiscovery` – Eureka, Consul, K8s DNS
- `14_APIGatewayPatterns` – Auth, rate limit, routing
- `15_ResiliencePatterns` – Circuit breaker states, bulkhead
- `16_DataConsistency` – Saga, Outbox, 2PC limitations
- `17_MessageQueue` – At-least-once vs exactly-once delivery

### System Design (15 questions)
- `18_DesignChatApp` – WebSockets, message storage
- `19_DesignPaymentGateway` – Idempotency, fraud detection
- `20_DesignJobScheduler` – Distributed cron, leader election

### Behavioral (10 questions)
- `21_STARMethod` – Situation, Task, Action, Result format
- `22_SeniorEngineerMindset` – Decision making, trade-offs
