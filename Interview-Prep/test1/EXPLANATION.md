# Test 1: Core Java & OOP Concepts

## 🎯 1. Overview
This module covers the "Big 4" of OOPs and the most common Java interview questions regarding memory, strings, and inheritance.

## 🍱 2. Questions in this Session:
1. **Encapsulation**: Private fields + Public getters.
2. **Inheritance**: Code reusability via `extends`.
3. **Polymorphism**: Compile-time (Overloading) vs Runtime (Overriding).
4. **Abstraction**: Interface vs Abstract Class.
5. **Static Behavior**: Method hiding, no overriding for static.
6. **Comparison**: `==` vs `.equals()`.
7. **The Final Keyword**: Immutable classes and constants.
8. **Constructor Chaining**: `super()` and `this()`.
9. **String Interning**: The String Constant Pool.
10. **The Diamond Problem**: Multiple inheritance in interfaces.

## 💡 3. Senior Tip
Always mention **Design Principles** (SOLID) when talking about OOPs. For example, Inheritance favors "Is-A", but Composition favors "Has-A".
