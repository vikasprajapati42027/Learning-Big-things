# Test 69: Couchbase

The high-performance JSON database.

## Unique Feature: N1QL
Couchbase allows you to use SQL-like language (**N1QL**) to query JSON documents. This makes it very easy for SQL developers to transition to NoSQL.

## Integrated Cache:
It has a built-in caching layer managed by the database itself, so you don't always need an external Redis cluster.
