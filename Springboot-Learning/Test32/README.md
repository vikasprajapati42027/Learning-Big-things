# Test 32: Aspect Oriented Programming (AOP)

Cross-cutting concerns are tasks that apply to many parts of your app.

## Examples:
- Logging
- Security checks
- Transaction management
- Performance monitoring

## Key Terms:
- **Aspect**: The class containing the cross-cutting code.
- **Advice**: The action taken (e.g., `@Before`).
- **Pointcut**: The expression that defines WHERE the advice applies.
- **Join Point**: The specific execution point (e.g., method call).
