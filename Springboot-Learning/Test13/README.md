# Test 13: Lombok (Boilerplate Reduction)

Lombok is a "must-have" for any Spring Boot project.

## Key Annotations:
- `@Getter` / `@Setter`: Generates getters and setters.
- `@ToString`: Generates a pretty `toString()`.
- `@NoArgsConstructor` / `@AllArgsConstructor`: Generates constructors.
- `@Data`: A bundle that includes all the above.
- `@Slf4j`: Generates a logger field `log`.
