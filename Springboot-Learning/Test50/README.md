# Test 50: WebSockets (Real-time)

Create interactive, alive applications.

## Use Cases:
1. **Chat Apps**: Instant messaging.
2. **Live Dashboards**: Stock prices or server metrics.
3. **Gaming**: Direct interaction between players.

## How it works:
1. Client starts with HTTP (Handshake).
2. Connection is "Upgraded" to a persistent TCP connection.
3. Both server and client can push data at any time without waiting for a request.
