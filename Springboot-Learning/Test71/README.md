# Test 71: Spring AI

The future of Java development is AI-enabled.

## Core Concepts:
- **AiClient**: The primary interface for interacting with LLMs.
- **Prompt**: The input message for the AI.
- **Response**: The AI's answer.
- **Output Parser**: Converts the AI's "chatty" text into a structured Java object/DTO automatically.

## Why Spring AI?
It provides a **Portable API**. You can write your logic once, and switch from OpenAI to Azure OpenAI just by changing your `application.properties`!
