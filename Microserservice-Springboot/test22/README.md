# Test 22: Kubernetes (K8s) Basics

Scaling to millions of users.

## Concept:
Docker runs containers. Kubernetes **manages** Docker.

## Key Features:
1. **Self-Healing**: If a microservice crashes, Kubernetes automatically starts a new one in 1 second.
2. **Auto-Scaling**: If your website gets a traffic spike, Kubernetes adds 10 more servers automatically.
3. **Service Discovery**: K8s has its own discovery system (it can replace Eureka!).

It's the industry standard for running microservices in the cloud.
