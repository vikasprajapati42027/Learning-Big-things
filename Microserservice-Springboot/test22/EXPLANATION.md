# Test 22: Kubernetes (K8s) Basics

Scaling and managing containers at scale.

## Key Files:
- `deployment.yaml`: Defines how many copies of the app should run and which image to use.
- `EXPLANATION.md`: Self-healing and Load Balancing in K8s.

## Why use it?
Kubernetes is the "Operating System" for the cloud. It handles the hardware for you. If a server dies, K8s moves your apps to a different server automatically.
