# Test 109: Vector Databases & RAG

Giving your AI "Long-Term Memory" using semantic search.

## Key Files:
- `KnowledgeService.java`: Shows how to store and search documents in a Vector Database (like Pinecone or Milvus).
- `EXPLANATION.md`: Understanding Embeddings and RAG (Retrieval-Augmented Generation).

## Concept:
A Vector database doesn't store words; it stores the "meaning" of sentences as a list of numbers (Vectors). This allows you to search for "citrus fruits" and find "Oranges" even if the word "citrus" isn't in the text.
