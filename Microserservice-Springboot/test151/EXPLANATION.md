# Test 151: The 100 Annotations Masterclass 🎓

A complete encyclopedia of Spring Boot annotations used by experts.

## Project Structure:
- `model/AnnotationDetail.java`: The data structure for our catalog.
- `controller/AnnotationMasterController.java`: API exposing the knowledge base.
- `config/`: Specialized config classes for each annotation type.
- `service/`: Demo services showing annotations in action.

## Categories Covered:
1. **Core Annotations**: @Bean, @Configuration, @Profile, @Conditional...
2. **Web Framework**: @PathVariable, @CookieValue, @ModelAttribute...
3. **Data/JPA**: @Transient, @PrePersist, @EmbeddedId...
4. **Resilience**: @Retryable, @Recover, @CircuitBreaker...
5. **Testing**: @Mock, @Spy, @Captor, @Tag...

## Data Accessibility:
Run the app and visit `/annotations` to see the full list with code snippets!
