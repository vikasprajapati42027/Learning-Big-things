# Test 108: Spring AI (Generative AI Integration)

Incorporating Large Language Models (LLMs) into your Java applications.

## Key Files:
- `AiController.java`: Demonstrates the unified `ChatClient` interface for OpenAI, Azure, or Ollama.
- `EXPLANATION.md`: Why you shouldn't call LLM APIs directly.

## Concept:
Spring AI is like "Spring Data" but for AI. It provides a standard way to talk to any AI model. If you want to switch from OpenAI to Meta's Llama, you just change a dependency—your code stays the same!
