# Test 49: Value Object

## 💎 1. Simple Explanation
An object that is defined by its **Values**, not its identity.
- **Example**: Money ($100).
- If I have two $100 bills, they are the same value. I don't care which serial number they have.
Value objects are always **Immutable** (cannot be changed). If you add $5, you get a NEW $105 object.
